Speed

  1. Press any "W", "A", "S","D" or Arrow keys to start
  2. You can use "WASD" or "Arrow" to control the snake
  3. There are 4-level of speed.
    • "0.5x": snake will constantly move with the half speed of the normal level
    • "1.0x": snake constantly move with the normal speed of the normal level. It's also the default level
    • "2.0x": snake constantly move with the double speed of the normal level
    • "Auto" level will auto increase the speed of snake time by time. The highest speed will go over 2.0x! Try and challenge yourself!
  4. The snake can't do 180degree turns.
